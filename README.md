# TestingJS

> This project is the first approach testing Javascript with JEST.

## Built With

- Javascript
- JEST

## Installation

1. Clone the repository in your local folder.
```
git clone https://github.com/rociac/testingJS.git
```
2. Run
```
npm install
```
```
npm run test
```
